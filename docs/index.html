Seja bem vindo!
Aprenda linguagem R com exemplos de código e exercícios!

R pode ser descrito como uma linguagem de programação e um ambiente de desenvolvimento para análises computacionais estaísticas e grafícos. R permite uma variedade de análises estatísticas e geração de gráficos, como:

  • Modelagem linear e não linear;
  • Testes estatísticos clássicos;
  • Análises Temporais;
  • Classificação;
  • Cluster, …

Como uma das principais características do R, é possível destacar a facilidade no processamento de grande volume de dados, assim como na geração de gráficos e no uso de símbolos matemáticos e equações. Cuidadosmanete pensada, a linguagem possui padrões para design de gráficos, permitindo o controle total do usuário, se necessário. R está disponível como Software Livre sob os termos da GNU General Software Foundation's Licença em forma de código fonte. Você pode executar R em uma grande variedade de plataformas UNIX e sistemas similares (incluindo FreeBSD e Linux), Windows e MacOS.
